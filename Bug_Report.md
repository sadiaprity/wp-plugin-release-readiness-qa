Bug ID: CF7-01
Title: Form submission fails without validation message
Severity: Medium
Environment: WordPress 6.x, PHP 8.x

Steps to Reproduce:
1. Open contact form
2. Leave required fields empty
3. Click submit

Expected Result:
User should see validation error message

Actual Result:
Form does not submit and no message is shown

Impact:
Users may think the form is broken and abandon the page
