# WordPress Plugin Release Readiness QA

This project demonstrates manual QA testing of a WordPress plugin
from a release-readiness perspective.

## Objective
To evaluate whether the plugin is stable, usable, and ready
for public release.

## Scope
- Installation & activation
- Core functionality
- User workflows
- Usability testing
- Compatibility testing
- Regression testing

## Tools Used
- WordPress (Local setup)
- Chrome, Firefox
- Chrome DevTools
- Manual test documentation
