
## Scenario 1: First-Time User Creates a Contact Form

Steps:
1. Install and activate plugin
2. Navigate to Contact > Add New
3. Create a basic form
4. Save and publish form
5. Embed form on a page
6. Submit form as user

Expected Result:
User can create and submit form without errors or confusion

## Scenario 2: Submit Form with Missing Required Fields

Steps:
1. Open contact form
2. Leave required fields empty
3. Submit form

Expected Result:
Clear validation message should be displayed
